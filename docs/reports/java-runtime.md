# Java Runtime

Shows information about the Java runtime (JRE or JDK) used to run JarHC. Unlike
the other sections, this section is a short text block rather than a table.

It shows the following information:

**Java home**

The installation directory of the Java runtime.

**Java runtime**

The name of the Java runtime.

**Java version**

The version of the Java runtime.

**Java vendor**

The vendor of the Java runtime.
